# 模板写作法完整框架参考

> 来源：《模板写作法》（山口拓朗）

---

## 一、三种基础写作模板

### 1. 列举型模板

**结构**：
1. **A. 传达整体构思** - 告知主题和列举要点数量
2. **B. 列举要点 1** - 第一个要点
3. **C. 列举要点 2** - 第二个要点
4. **D. 列举要点 3** - 第三个要点（可根据实际情况增减）
5. **E. 总结** - 概括要点，或升华主题

**模板**：
```
【整体构思】
[主题] 有 [数量] 个 [要点类型]。

【要点一】
[具体内容]

【要点二】
[具体内容]

【要点三】
[具体内容]

【总结】
[概括或升华]
```

**适用场景**：
- 整理并列信息
- 让文章条理清晰
- 无压力阅读

**示例框架**：
```
碳酸水有 3 个优点。

要点 1：入口爽快感
要点 2：防止暴食
要点 3：缓解疲劳

总结：碳酸水有益身心健康。
```

---

### 2. 结论优先型模板

**结构**：
1. **A. 推出结论** - 开门见山，亮出核心观点
2. **B. 阐述理由·依据** - 提供支撑结论的逻辑或事实依据
3. **C. 阐述具体事例·详细内容** - 用案例、细节或亲身经历增强真实感与说服力
4. **D. 总结** - 重申或强调结论

**模板**：
```
【结论】
我的结论是：___________

【理由】
理由是：
- 第一，___________
- 第二，___________

【事例】
例如：___________

【总结】
因此，我重申：___________
```

**适用场景**：
- 重点传达核心观点
- 提升文章说服力
- 获得读者认同

**示例框架**：
```
结论：我每天散步 30 分钟。

理由：适度流汗能焕活身心。

事例：坚持散步后，压力减小，不易感冒。

总结：今后会坚持。
```

---

### 3. 故事型模板

**结构**：
1. **A. 阐述消沉状态** - 描述主人公（可以是自己）最初的困境、烦恼或负面状态
2. **B. 阐述转机** - 交代带来改变的契机（人、事、物、观念等）
3. **C. 阐述转变与成长** - 描述转变的具体过程和积极变化
4. **D. 展望未来** - 表达对未来的期望或总结从中获得的启示

**模板**：
```
【消沉状态】
以前，我___________

【转机】
直到有一天/直到我遇到___________

【转变与成长】
从那以后，我___________

【展望未来】
今后/希望___________
```

**适用场景**：
- 通过讲述经历引发读者共鸣
- 打动人心
- 分享个人成长

**示例框架**：
```
消沉状态：以前最不擅长整理收纳。

转机：遇到《怦然心动的人生整理术》这本书。

转变与成长：学会只保留心动物品，身边焕然一新。

展望未来：听从心声，做好整理收纳。
```

## 二、进阶组合模板

### 组合①：故事型 + 结论优先型

**结构**：
```
A. 消沉状态：[描述困境]
B. 转机：（此处插入结论优先型）
   └─ B-a. 结论：[学会的方法]
   └─ B-b. 理由：[方法的原理]
   └─ B-c. 事例：[具体做法]
C. 转变与成长：[实践效果]
D. 展望未来：[推荐给大家]
```

**适用场景**：
- 在故事叙述中详细解释某个方法
- 在"转机"部分插入结论优先型

**完整示例**：
```
文章：《新型预防中暑的方法——"冷却手掌心"的奇特效果》

A. 消沉状态：深受中暑困扰
B. 转机：（插入结论优先型）
   - 结论：学会"冷却手心"法
   - 理由：手心和脚底是散热窗口，冷却可降低核心体温
   - 事例：手握冰镇水瓶即可
C. 转变与成长：实践后有效预防中暑
D. 展望未来：推荐给大家
```

---

### 组合②：结论优先型 + 故事型

**结构**：
```
A. 结论：[核心观点]
B. 理由：[逻辑依据]
C. 事例：（此处插入故事型）
   └─ C-a. 消沉状态：[过去的困境]
   └─ C-b. 转机：[改变的契机]
   └─ C-c. 转变与成长：[积极变化]
   └─ C-d. 展望未来：[获得的启示]
D. 总结：[重申结论]
```

**适用场景**：
- 在逻辑论证中插入亲身经历
- 用故事作为有力例证

---

### 组合③：结论优先型 + 列举型 + 故事型

**结构**：
```
A. 结论：[核心观点]
B. 阐述理由·依据：（此处使用列举型）
   └─ 理由 1：[第一点]
   └─ 理由 2：[第二点]
   └─ 理由 3：[第三点]
C. 阐述具体事例·详细内容：（此处使用故事型）
   └─ 消沉状态：[过去的困境]
   └─ 转机：[改变的契机]
   └─ 转变与成长：[积极变化]
   └─ 展望未来：[获得的启示]
D. 总结：[展望效果]
```

**适用场景**：
- 完整论证 + 分点说明 + 感染力
- 需要充分说服力的场合

**完整示例**：
```
文章：《关于在办公室导入"5S"管理法的提议》

A. 结论：提议导入"5S"法

B. 阐述理由·依据：（列举型）
   - 理由 1：提升效率
   - 理由 2：减少失误
   - 理由 3：改善形象

C. 阐述具体事例·详细内容：（故事型）
   - 消沉状态：仓库从混乱到有序的故事
   - 转机：学习 5S 管理法
   - 转变与成长：实施后的积极变化
   - 展望未来：对办公室的期望

D. 总结：展望导入后的积极效果
```

## 三、写作前必须确认的 5 项内容（清单）

在动笔前，明确以下五点，能显著提升写作速度与质量：

| 序号 | 检查项 | 说明 |
|------|--------|------|
| 1 | **目的** | 写这篇文章是为了什么？（告知、说服、请求、共鸣等） |
| 2 | **读者** | 文章是写给谁看的？（明确具体对象） |
| 3 | **需求** | 读者内心渴望或需要什么信息？ |
| 4 | **期望反应** | 希望读者读后产生怎样的反应或行动？ |
| 5 | **知识水平** | 读者的相关知识水平和理解能力如何？ |

## 四、关键写作技巧

### 1. 一行概括

动笔前，尝试用一行字概括全文及各部分核心，确保思路清晰。

**示例**：
- 全文概括："碳酸水有 3 个优点"
- 各部分概括："要点 1 是入口爽快感"

### 2. 抽象与具体转换

在写作中交替使用抽象概念（观点、总结）和具体描述（细节、案例），帮助读者理解和认同。

**示例**：
- 抽象："休闲设施"
- 具体："健身房、游泳池、桑拿房"

### 3. 自问自答

模板的每个部分都隐含了一个需要回答的问题，顺着问题填充内容即可。

**示例问题**：
- "碳酸水有几个优点？" → 3 个
- "第一个优点是什么？" → 入口爽快感
- "为什么这么说？" → 因为...

## 五、常用句式模板

### 开头句式

| 场景 | 句式 |
|------|------|
| 列举型 | "[主题] 有 [数量] 个 [要点类型]。" |
| 结论优先型 | "我的结论是：[核心观点]。" |
| 故事型 | "以前的我，[消沉状态]。" |

### 连接句式

| 用途 | 句式 |
|------|------|
| 列举 | "第一，..."、"第二，..."、"第三，..." |
| 阐述理由 | "理由是..."、"之所以这么说，是因为..." |
| 举例 | "例如..."、"以...为例" |
| 转折 | "然而..."、"但是..." |
| 总结 | "综上所述..."、"总之..." |

### 结尾句式

| 场景 | 句式 |
|------|------|
| 列举型 | "综上所述，[主题] 的 [要点数量] 个 [要点类型] 是..." |
| 结论优先型 | "因此，我再次强调：[结论]。" |
| 故事型 | "今后，我将继续..." |
