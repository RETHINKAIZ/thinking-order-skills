# 模板写作法可套用模板库

> 基于《模板写作法》（山口拓朗）整理

---

## 一、列举型模板

### 基础列举模板

```
【标题】[主题]

【整体构思】
[主题] 有 [数量] 个 [要点类型]。

【要点一】
[要点 1 的标题]
[具体内容描述]

【要点二】
[要点 2 的标题]
[具体内容描述]

【要点三】
[要点 3 的标题]
[具体内容描述]

【总结】
[概括要点或升华主题]
```

### 优缺点列举模板

```
【标题】[事物] 的优缺点分析

【整体构思】
[事物] 有 [数量] 个优点和 [数量] 个缺点。

【优点】
1. [优点 1]：[具体说明]
2. [优点 2]：[具体说明]
3. [优点 3]：[具体说明]

【缺点】
1. [缺点 1]：[具体说明]
2. [缺点 2]：[具体说明]

【总结】
[充分考虑利弊后的建议]
```

### 检查清单模板

```
【标题】[事项] 检查清单

【整体构思】
[事项] 前，请确认以下 [数量] 项内容。

【检查项目】
- [ ] [检查项 1]
- [ ] [检查项 2]
- [ ] [检查项 3]
- [ ] [检查项 4]
- [ ] [检查项 5]

【补充说明】
[对部分检查项的详细解释]

【总结】
[完成检查后的建议]
```

## 二、结论优先型模板

### 基础结论优先模板

```
【标题】[主题]

【结论】
我的结论是：[核心观点]

【理由】
支持这一结论的理由有：

理由一：[理由 1 的标题]
[具体说明]

理由二：[理由 2 的标题]
[具体说明]

【事例】
[具体案例/亲身经历/数据支撑]

【总结】
[重申结论或呼吁行动]
```

### 建议提案模板

```
【标题】关于 [事项] 的建议

【结论】
我建议：[核心建议]

【理由】
理由是：
1. [理由 1]
2. [理由 2]
3. [理由 3]

【事例】
以 [公司/案例] 为例：
[具体说明]

【回应反驳】
[预判可能的反对意见并回应]

【总结】
[重申建议的重要性]
```

### 观点表达模板

```
【标题】[观点]

【观点】
我的观点是：[核心观点]

【理由】
为什么这么说？
1. [理由 1]
2. [理由 2]

【事例】
[具体案例或数据]

【总结】
[强调观点]
```

### 加入背景的结论优先模板

```
【标题】[主题]

【背景】
[描述当前情况或背景]

【结论】
因此，我建议：[核心结论]

【理由】
理由是：
1. [理由 1]
2. [理由 2]

【事例】
[具体案例]
- 案例 1：[说明]
- 案例 2：[说明]

【总结】
[呼吁行动或展望]
```

## 三、故事型模板

### 基础故事模板

```
【标题】[故事主题]

【消沉状态】
以前的我/当时，[描述困境、烦恼或负面状态]
[当时的心情/处境]

【转机】
直到有一天/直到我遇到 [契机]
[这个人/这件事/这本书] 改变了我的想法

【转变与成长】
从那以后，我 [描述转变的具体过程]
[积极变化的具体表现]

【展望未来】
今后，我 [表达对未来的期望]
[总结从中获得的启示]
```

### 成长经历模板

```
【标题】[成长主题]

【消沉状态】
[时间]，我 [描述当时的状态]
- 面临的困难：[具体说明]
- 心情：[感受描述]

【转机】
[时间点]，我 [遇到的契机]
- 契机的来源：[人/事/物]
- 带来的冲击：[感受]

【转变与成长】
之后，我 [采取的行动]
- 行动 1：[说明]
- 行动 2：[说明]
- 取得的变化：[具体成果]

【展望未来】
这段经历让我明白 [获得的启示]
今后，我将继续 [未来的期望]
```

### 软文故事模板

```
【标题】[产品/服务] 改变了我的生活

【消沉状态（担忧）】
之前，我为 [问题] 而烦恼
- 尝试过的方法：[说明]
- 效果：[不理想的原因]

【转机】
直到我遇到了 [产品/服务名称]
- 第一印象：[说明]
- 决定尝试的理由：[说明]

【转变与成长（成效）】
使用后，[具体变化]
- 变化 1：[说明]
- 变化 2：[说明]
- 现在的状态：[描述]

【展望未来】
今后，我 [对美好生活的期望]
[推荐给大家]
```

### 书评/读后感模板

```
【标题】《[书名]》读后感

【消沉状态】
读这本书之前，我 [当时的状态/困惑]
- 困扰我的问题：[说明]

【转机】
遇到这本书的契机
- [如何遇到这本书]
- 书中的核心观点：[说明]

【转变与成长】
读完这本书后，我 [获得的启发]
- 观念改变：[说明]
- 行动改变：[说明]

【展望未来】
这本书让我明白 [核心启示]
推荐给 [目标读者]
```

## 四、组合模板

### 组合①：故事型 + 结论优先型

```
【标题】[主题]

【消沉状态】
[描述最初的困境]

【转机】
[遇到契机后，学会了一个方法]

   【结论】
   这个方法是：[核心方法]

   【理由】
   为什么有效？
   1. [原理 1]
   2. [原理 2]

   【事例】
   具体做法：[操作步骤]

【转变与成长】
实践这个方法后，[取得的成效]

【展望未来】
推荐给 [目标人群]
```

### 组合②：结论优先型 + 故事型

```
【标题】[主题]

【结论】
[核心观点]

【理由】
[逻辑论证]

【事例】（故事型）
   【消沉状态】
   [过去的困境]

   【转机】
   [改变的契机]

   【转变与成长】
   [积极变化]

   【展望未来】
   [获得的启示]

【总结】
[重申结论]
```

### 组合③：结论优先型 + 列举型 + 故事型

```
【标题】[主题]

【结论】
[核心观点/建议]

【理由】（列举型）
理由有 [数量] 点：

1. [理由 1 的标题]
   [具体说明]

2. [理由 2 的标题]
   [具体说明]

3. [理由 3 的标题]
   [具体说明]

【事例】（故事型）
   【消沉状态】
   [描述过去的困境]

   【转机】
   [遇到的契机]

   【转变与成长】
   [发生的积极变化]

   【展望未来】
   [获得的启示]

【总结】
[展望实施后的效果]
```

## 八、快速选择指南

| 场景 | 推荐模板 | 模板位置 |
|------|----------|----------|
| 列举多个要点 | 列举型基础模板 | 一、列举型模板 |
| 分析优缺点 | 优缺点列举模板 | 一、列举型模板 |
| 检查清单 | 检查清单模板 | 一、列举型模板 |
| 提案建议 | 建议提案模板 | 二、结论优先型模板 |
| 观点表达 | 观点表达模板 | 二、结论优先型模板 |
| 分享经历 | 基础故事模板 | 三、故事型模板 |
| 成长故事 | 成长经历模板 | 三、故事型模板 |
| 产品软文 | 软文故事模板 | 三、故事型模板 |
| 书评读后感 | 书评模板 | 三、故事型模板 |
| PR 文章 | PR 文章模板 | 五、邮件/公文模板 |
| 会议报告 | 会议报告模板 | 五、邮件/公文模板 |
| 博客文章 | 博客文章模板 | 七、博客/文章模板 |
