# 参考资料

- **CaseReference.md** - 《模板写作法》全书全部案例参考

## 使用说明

本技能基于《模板写作法》（山口拓朗）一书整理，提供：

1. **三种基础模板**：列举型、结论优先型、故事型
2. **三种组合模板**：故事型 + 结论优先型、结论优先型 + 故事型、结论优先型 + 列举型 + 故事型
3. **完整案例库**：全书 21 篇完整例文 + 数十个片段示例

使用技能时，会自动匹配场景推荐模板，并通过交互式提问引导用户完成内容填充。
