### 二、结论优先型模板案例

#### 例 1：每天散步 30 分钟（第 1 章，基础示例）

```
【结论】
我每天散步 30 分钟。

【理由】
适度流汗能焕活身心。

【事例】
坚持散步后，压力减小，不易感冒。

【总结】
今后会坚持。
```

---

#### 例 2：睡前不看手机（第 3 章，分解示例）

```
【结论】
睡觉前一小时尽量不看手机。

【理由】
手机光会使交感神经活跃，影响睡眠。

【事例】
改看书后，睡眠质量改善。

【总结】
坚持"睡前戒手机"。
```

#### 例 3：建议给孩子读绘本（第 1 章，自问自答示例）

```
问：为什么建议给孩子读绘本？
答：因为能促进亲子关系，培养语言能力。

问：具体有什么好处？
答：1. 增进亲子交流 2. 提高词汇量 3. 培养想象力

问：有什么证据？
答：研究表明，经常被读绘本的孩子语言能力更强。

结论：建议每天给孩子读绘本。
```

#### 例 6：异议书《我反对》（第 3 章，完整例文）

```

我反对降价策略。

【理由】
1. 降价会损害品牌价值
2. 吸引对价格敏感的非目标客户
3. 长期来看不利于利润

【事例】
某品牌降价后，虽然短期销量上升，但品牌形象受损。

【回应反驳】
理解降价诉求，但阐明弊大于利。

【总结】
建议采用提升价值而非降价的策略。
```

## 第二部分：组合模板案例

### 例 1：组合①"故事型 + 结论优先型"

**文章**：《新型预防中暑的方法——"冷却手掌心"的奇特效果》

```
【消沉状态】
深受中暑困扰，每年都难受。

【转机】（插入结论优先型）

   【结论】
   学会"冷却手心"法。

   【理由】
   手心和脚底是散热窗口，冷却可降低核心体温。

   【事例】
   手握冰镇水瓶即可。

【转变与成长】
实践后有效预防中暑。

【展望未来】
推荐给大家。
```

---

### 例 2：组合③"结论优先型 + 列举型 + 故事型"

**文章**：《关于在办公室导入"5S"管理法的提议》

```
【结论】
提议导入"5S"管理法。

【阐述理由·依据】（列举型）
理由有 3 点：

1. 提升效率
   整理整顿后，找东西时间大大减少

2. 减少失误
   清晰标识后，拿错东西的情况几乎消失

3. 改善形象
   整洁的办公环境给客户留下好印象

【阐述具体事例·详细内容】（故事型）

   【消沉状态】
   之前仓库混乱，经常找不到东西

   【转机】
   学习并导入 5S 管理法

   【转变与成长】
   仓库从混乱到有序，效率大幅提升

   【展望未来】
   办公室也能实现同样效果

【总结】
展望导入后的积极效果。
```

## 第四部分：常用句式示例

### 开头句式

| 用途 | 句式 |
|------|------|
| 列举型开头 | "[主题] 有 [数量] 个 [要点类型]。" |
| 结论优先型开头 | "我的结论是：[核心观点]。" |
| 故事型开头 | "以前的我，[消沉状态]。" |
| 背景引入 | "关于 [事项]，目前的情况是..." |

### 连接句式

| 用途 | 句式 |
|------|------|
| 列举连接 | "第一，..."、"第二，..."、"第三，..." |
| 阐述理由 | "理由是..."、"之所以这么说，是因为..." |
| 举例说明 | "例如..."、"以...为例"、"具体来说..." |
| 转折 | "然而..."、"但是..."、"不过..." |
| 总结 | "综上所述..."、"总之..."、"因此..." |

### 结尾句式

| 用途 | 句式 |
|------|------|
| 列举型结尾 | "综上所述，[主题] 的 [数量] 个 [要点] 是..." |
| 结论优先型结尾 | "因此，我再次强调：[结论]。" |
| 故事型结尾 | "今后，我将继续..." |
| 呼吁行动 | "推荐大家尝试..."、"不妨从...开始" |
