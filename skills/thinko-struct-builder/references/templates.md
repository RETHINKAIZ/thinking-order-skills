### AIDA 推广模板

```
【Attention 吸引注意】
开场钩子：
□ 提问：您是否遇到过___________？
□ 数据：___________的人都在为___________烦恼
□ 故事：上个月，一位客户告诉我___________

【Interest 引起兴趣】
解决方案亮点：
- ___________
- ___________
- ___________

【Desire 激发欲望】
使用后能带来的改变：
- 现状：___________
- 使用后：___________
- 对比：___________

【Action 促成行动】
行动号召：
- 限时优惠：___________
- 零风险承诺：___________
- 立即行动：___________
```

### SPIN 提问模板

```
【Situation 现状问题】
了解客户当前情况：
- 您目前是如何处理___________的？
- 您使用的是什么系统/方法？
- 团队规模是？

【Problem 难点问题】
挖掘客户痛点：
- 这个过程中最麻烦的是什么？
- 您遇到过什么问题？
- 哪些方面让您不满意？

【Implication 暗示问题】
放大痛点影响：
- 这个问题会导致什么后果？
- 如果不解决，会影响___________吗？
- 这给团队/公司带来的损失是？

【Need-payoff 解决型问题】
引导客户说出需求：
- 如果能解决这个问题，对您意味着什么？
- 您希望实现什么样的效果？
- 您认为理想的方案应该是怎样的？
```

- 第三，___________

## 写作模板

### SCQA 文章序言模板

```
【Situation 情境】
（背景描述，读者已知的信息）
我们都知道/目前的情况是：
___________

【Complication 冲突】
（打破平衡的事件/问题）
然而/但是/最近：
___________

【Question 疑问】
（自然引出的问题）
那么/我们应该如何：
___________

【Answer 回答】
（文章的核心观点/方案）
我们的答案是：
___________

（正文开始...）
```
